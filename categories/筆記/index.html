JavaScript Scope Chain 與 Closure

Scope 到底是指哪裡？

Scope 是變數的有效範圍。

function 裡面是 local，指的是在 local 這個範圍裡面有效。

function 外面是 global，指的是在 global 這個範圍裡面有效。
