JavaScript 變數作用域 Variable Scope

變數的作用域（Scope）

作用域是指變數的有效範圍。

常聽到的全域變數（global variable）通常是指：全域變數是不在任何function(){}內的變數，換句話說，全域變數是在任何地方都可以用的變數。

跟全域變數相反，function(){}內的變數稱為區域變數（local variable）。
